## Проектная работа с с интеграцией через WEB - сервисы двумя методами.

В данной проектной работе реализована интеграция основной базы с мобильным приложением. Конечно проект учебный, поэтому интеграции реализованы в целях понимания процесса обмена данными.

В целом работа включает в себя реализацию основных задач оперативного и бухгалтерского учёта, но это реализовано по минимуму. Основной акцент в данной работе я сделал на создании моей первой в жизни интеграции.
Причём, сначала реализовал через метод POST, а оказалось, что в техзадании требуется реализация обмена через метод GET. Что конечно показалось странным, так как параметры с именем пользователя и телефоном приходится передавать в тексте строки.

Принцип работы мобильного приложения:
* Возможность регистрации нового пользователя. Происходит создание нового пользователя ИБ с полными правами, а также создание нового клиента в справочнике "Контрагенты" и справочнике "Пользователи".
* Вход под уже существующим в основной базе пользователем. 
* Запись на услугу в мобильном приложении. В основной базе создается документ "Предварительная запись", где пользователь является клиентом.

Работа выполнена в 1С: предприятие 8.3, версии для разработчиков.
